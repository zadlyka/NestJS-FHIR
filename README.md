## Description

[NestJS-FHIR](https://github.com/zadlyka/NestJS-FHIR) Using NestJS, MongoDB and FHIR HL7 v4.3.0

## Installation

```bash
$ npm install
```
## Configuration
```bash
rename the .env.example file to .env, and configure it.
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
